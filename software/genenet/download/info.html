Example Sessions:

These R scripts provide some examples how to use GeneNet with real data:

  • Arabidopsis thaliana network of diurnal cycle genes from Smith et al. 2004 expression data
  • Ecoli network (from Schaefer and Strimmer, SAGMB 2005)
  • T-cell network (from Opgen-Rhein and Strimmer, WCSB 2006)
